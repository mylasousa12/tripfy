# ✈️ Tripfy

Tripfy é uma plataforma web de turismo desenvolvida para facilitar a experiência de viajantes, permitindo consultar moedas, encontrar pontos turísticos e localizar serviços próximos de acordo com a localização do usuário.

O projeto está sendo desenvolvido com foco em performance, escalabilidade, responsividade e experiência do usuário.

---

## 🚀 Funcionalidades

- 🔐 Autenticação JWT
- 👤 Cadastro e login de usuários
- 🔑 Recuperação e redefinição de senha
- 🌎 Consulta de moedas e conversão de valores
- 📍 Busca de pontos turísticos
- 🍽️ Localização de restaurantes, mercados, farmácias e outros serviços
- 📱 Interface responsiva
- ♿ Aplicação de acessibilidade e HTML semântico
- 🔄 Integração com APIs REST
